# Agent Rules — convergio-sdk

Tool-agnostic. Claude Code, Cursor, Copilot, Windsurf, any AI tool.

## This repo

SDK foundation for all Convergio crates: types, telemetry, db, security.
Changes here affect 36+ downstream crates. Be careful.

```
crates/
├── convergio-types/      — Extension trait, Manifest, DomainEvent, ApiError
├── convergio-telemetry/  — tracing, health registry, metrics
├── convergio-db/         — r2d2 SQLite pool, migration runner
└── convergio-security/   — JWT, AEAD, RBAC, audit, trust, sandbox, SSRF
```

Dependency graph: telemetry → types, db → types, security → types + db

## Build

```bash
cargo fmt --all -- --check
RUSTFLAGS="-Dwarnings" cargo clippy --workspace --all-targets --locked
cargo test --workspace --locked
cargo deny check                   # license + dependency policy
```

Zero tolerance: 0 warnings, 0 skipped tests, 0 TODO without issue link.

## Rules

- English only (code + docs)
- Max 250 lines per file
- Conventional commits: `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`
- Breaking changes = minor version bump (pre-1.0)
- PR: 5 sections (Problem, Why, What changed, Validation, Impact)

## Test requirements

| Type | Location | Required |
|------|----------|----------|
| Unit | `src/*.rs` `#[cfg(test)]` | Always |
| Integration | `tests/integration_*.rs` | Always |
| Adversarial | `tests/adversarial_*.rs` | Always (this is the security crate) |

Coverage: ≥80%. No hardcoded counts, no hardcoded versions.

## Comments — only 2 types

- `//!` module doc — 1-2 lines: what, who uses it
- `// WHY:` — non-obvious choice explanation

Forbidden: trivial doc comments, code-repeating comments, `/// # Examples`.

## Documentation

- `docs/adr/` — one ADR per non-obvious decision
- README.md — always current with crate table and manifesto
- CHANGELOG.md — auto-generated by release-please

## Do NOT

- Ship without unit + integration + adversarial tests
- Merge without user approval (auto-merge handles CI-green PRs)
- Leave warnings, dead code, or TODO without issue
- Add deps without checking license compatibility (deny.toml)
